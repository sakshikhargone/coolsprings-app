## Setup  

1. **Clone the repository:**  
   ```bash
   git clone https://https://github.com/sakshikhargone/coolsprings-app.git
   ```  

2. **Navigate to the project directory:**  
   ```bash
   cd coolsprings-app
   ```  

3. **Install dependencies:**  
   ```bash
   npm install
   ```  

4. **Start the development server:**  
   ```bash
   npm run start
   ```
